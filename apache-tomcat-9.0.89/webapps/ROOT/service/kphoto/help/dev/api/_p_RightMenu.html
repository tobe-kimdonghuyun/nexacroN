RightMenu

RightMenu에서 사용할 메뉴를 설정합니다.

remarks

기본값은 "figure, emoticon, addimage" 입니다.

"figure"는 도형, "emoticon"은 이모티콘, "addimage"는 이미지 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토의 RightMenu를 figure, emoticon으로 설정합니다.
            RightMenu: 'figure,emoticon'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
