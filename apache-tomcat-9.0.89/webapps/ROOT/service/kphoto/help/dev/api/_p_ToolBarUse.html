K Photo :: Config :: ToolBarUse

ToolBarUse

포토의 툴바를 표시하거나 숨기는 기능을 설정합니다.

remarks

기본값은 "1" 이고, "0" 으로 설정 시 툴바를 표시하지 않습니다.

ToolBar   툴바의 메뉴를 설정합니다.

툴바의 메뉴는
open : 열기
save : 저장
history : 되돌리기, 다시실행
rotate : 좌/우로 회전
select : 선택
flip : 좌/우, 상/하 반전
resize : 크기조절
size_in_decrease : % 늘리기, 줄이기
tone : 보정
filter : 필터
fullscreen : 전체화면
text : 텍스트 추가
align : 정렬
separator : 구분선

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토의 툴바를 표시합니다.
            ToolBarUse: '1',

            // 포토의 메뉴를 "열기,구분선,저장,구분선,보정" 으로 설정합니다.
            ToolBar: 'open,separator,save,separator,tone'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
