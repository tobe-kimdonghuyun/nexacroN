K Photo :: OnError

void RAONKPHOTO_OnError(photoID, paramObj)

에러가 발생할 경우 발생합니다.

return value

없음.

parameters

photoID  에러가 발생한 포토의 id를 의미합니다.
paramObj  에러에 대한 정보가 있는 object 입니다.
- paramObj.strCode  오류코드를 의미합니다.
- paramObj.strMessage  오류 메시지를 의미합니다.

remarks

없음.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	
<script type="text/javascript">

    function RAONKPHOTO_OnError(photoID, paramObj){
        // 에러 발생 후 오류메시지 띄워주는 예제
        alert("Error Code : " + paramObj.strCode + "\nError Message : " + paramObj.strMessage);
    }

</script>
