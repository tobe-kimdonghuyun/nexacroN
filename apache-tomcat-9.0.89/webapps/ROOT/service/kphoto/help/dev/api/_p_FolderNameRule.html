K Photo :: Config :: FolderNameRule

FolderNameRule

서버에서 저장할 파일의 폴더명을 지정하는 규칙입니다.

remarks

일반 폴더 또는 년월일폴더를 사용할 수 있습니다.

일반폴더는 "사용자 설정 값" 형태로 사용할 수 있습니다.
년월일폴더 설정 방법은
사용하지 않음 => "/",
년도 => "YYYY/",
년도/월 => "YYYY/MM/",
년도/월/일 => "YYYY/MM/DD/" 중 하나의 값을 사용합니다.

폴더경로는 "raonkphotodata" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            //파일을 저장할 폴더명을 년도/월 로 설정합니다.
            FolderNameRule : 'YYYY/MM/'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
