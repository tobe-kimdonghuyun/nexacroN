K Photo :: Config :: SaveMethod

SaveMethod

이미지 저장 방식을 설정합니다.

remarks

기본값은 "upload"이며, "base64"로 설정 시 서버로 저장하지 않고 base64 데이터로 리턴됩니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토의 업로드 방식을 upload로 설정합니다.
            SaveMethod: 'upload'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
