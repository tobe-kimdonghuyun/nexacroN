K Photo :: Config :: TextDefaultFontSize

TextDefaultFontSize

텍스트 메뉴의 기본 글자 크기를 설정합니다.

remarks

기본 글자 크기는 "14pt" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 텍스트 메뉴의 기본 글자 크기를 23 으로 설정합니다.
            TextDefaultFontSize: '23'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
