K Photo :: Config :: TextFontFamily

TextFontFamily

텍스트 메뉴의 기본 글꼴에 노출 할 폰트 리스트를 설정합니다.

remarks

기본 글꼴은 "맑은 고딕" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 텍스트 메뉴의 기본 글꼴에 노출 할 폰트를 맑은 고딕, 굴림, 궁서로 설정합니다.
            TextFontFamily: '맑은 고딕,굴림,궁서'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
