K Photo :: Config :: TextBackgroundBorderColor

TextBackgroundBorderColor

텍스트 객체의 배경 테두리 색을 설정합니다.

remarks

기본값은 "#000000" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 텍스트 객체의 배경 테두리 색을 #FF9900으로 설정합니다.
            TextBackgroundBorderColor: '#FF9900'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
