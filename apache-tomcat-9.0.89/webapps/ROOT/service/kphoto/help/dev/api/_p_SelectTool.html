K Photo :: Config :: SelectTool

SelectTool

이미지에서 영역을 선택했을 때 보여지는 메뉴입니다.

remarks

기본값은 "crop, mosaic" 입니다.

"crop"은 자르기, "mosaic"는 모자이크 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 영역 선택 시 보여지는 메뉴를 mosaic로 설정합니다.
            SelectTool: 'mosaic'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
