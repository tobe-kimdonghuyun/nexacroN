K Photo :: Config :: Lang

Lang

포토에 로드 할 언어를 설정합니다.

remarks

기본값은 "ko-kr : 한국어" 입니다.

언어코드는
ko-kr : 한국어(대한민국)
en-us : 영어(미국)
zh-cn : 중국어(중국)
zh-tw : 중국어(대만)
ja-jp : 일본어(일본)

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토 언어를 영어로 설정합니다.
            Lang: 'en-us'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
