K Photo :: Config :: HandlerUrl

HandlerUrl

웹에서 RFC 1867 표준에 따르는 POST 방식의 파일 전송을 받아주는 페이지 URL입니다.
서버에 파일이 업로드 되지 않을 경우 이 값의 설정을 확인하세요.

remarks

기본값은 "" 입니다.

kphoto/handler/ 아래의 각 언어에 맞는 페이지를 호출합니다.
상대 및 절대 경로 모두를 지원합니다.
각 개발 언어에 맞는 값으로 설정하세요.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 업로드 handler url을 설정합니다.
            HandlerUrl: 'http://www.raonk.com/handler/raonkhandler.jsp'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
