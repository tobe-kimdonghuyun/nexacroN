K Photo :: Config :: FileNameRule

FileNameRule

서버에서 저장할 파일의 이름을 지정하는 규칙입니다.

remarks

기본값은 "GUID" 입니다.

값 지정은 "GUID" 또는 "REALFILENAME" 을 입력합니다.
"GUID" 의 경우 파일명이 GUID값으로 설정됩니다.
"REALFILENAME" 의 경우 파일의 실제 파일명으로 설정됩니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            //저장할 파일명을 REALFILENAME 값으로 설정합니다.
            FileNameRule : 'REALFILENAME'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
