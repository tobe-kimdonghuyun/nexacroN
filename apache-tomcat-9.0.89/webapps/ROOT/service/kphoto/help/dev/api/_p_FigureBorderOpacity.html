K Photo :: Config :: FigureBorderOpacity

FigureBorderOpacity

도형의 선 투명도를 설정합니다.

remarks

기본값은 "100" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 도형의 선 투명도를 80으로 설정합니다.
            FigureBorderOpacity: '80'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
