K Photo :: Config :: PhotoHolder

PhotoHolder

포토를 특정 영역에서 생성할 수 있도록 합니다.

remarks

포토를 생성할 영역의 id값을 입력합니다.
필요시 PhotoHolder:"아이디값" 으로 설정합니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->

<div id="photo_area"></div>
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토를 생성할 영역의 ID를 설정합니다.
            PhotoHolder: 'photo_area'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
