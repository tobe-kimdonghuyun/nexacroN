K Photo :: Config :: FigureWidth

FigureWidth

도형의 넓이를 설정합니다.

remarks

기본값은 "50" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 도형의 넓이를 100으로 설정합니다.
            FigureWidth: '100'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
