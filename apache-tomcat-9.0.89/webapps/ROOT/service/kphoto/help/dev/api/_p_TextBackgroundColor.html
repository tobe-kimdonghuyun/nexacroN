K Photo :: Config :: TextBackgroundColor

TextBackgroundColor

텍스트 객체의 배경 색을 설정합니다.

remarks

기본값은 "#FFFFFF" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 텍스트 객체의 배경 색을 #FF3C3C으로 설정합니다.
            TextBackgroundColor: '#FF3C3C'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
