K Photo :: Config :: GetText

GetText

이미지에 추가된 텍스트를 뽑을지를 설정합니다.

remarks

기본값은 "1"이며, "0"으로 설정 시 이미지에 추가된 텍스트를 뽑지 않습니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<script type="text/javascript">
    function RAONKPHOTO_OnSaveComplete(photoID, returnObject){
        //이미지 저장 후 이미지에 추가 된 텍스트 보여주는 예제
        var text = "";

        if (returnObject.text) {
            var len = returnObject.text.length;
            for (var i = 0; i < len; i++){
                text += 'text ' + i + ' -> ' + returnObject.text[i] + '\n';
            }
        } else{
            text = "추가 된 텍스트가 없습니다.";
        }
        alert(text);
    }
</script>

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">

        };

    </script>
