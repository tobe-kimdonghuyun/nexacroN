K Photo :: Config :: Visibility

Visibility

포토에디터를 보이거나 숨깁니다.

remarks

기본값은 "show"이고, "hidden"으로 설정 시 포토에디터를 숨깁니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토에디터를 숨김으로 설정합니다.
            Visibility: 'hidden'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
