K Photo :: Config :: AddImageXmlUrl

AddImageXmlUrl

Right Menu에 있는 이미지를 로드하기 위한 xml 파일을 설정합니다.

remarks

기본값은 "raonkphoto.userimage.xml" 입니다.
파일 경로는 "kphoto/config" 입니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토의 Right메뉴에 들어가는 이미지를 example.xml에서 로드하여 생성합니다.
            AddImageXmlUrl: 'example.xml'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
