K Photo :: OnPopupEvent

void RAONKPHOTO_OnPopupEvent(photoID, paramObj)

포토에디터가 팝업형태로 띄워지거나 닫혔을 때 발생합니다.

return value

없음.

parameters

photoID  팝업형태인 포토에디터의 id를 의미합니다.
paramObj  팝업이 띄워지거나 닫힌것을 의미합니다.
- paramObj.value  값이 "show"인 경우 팝업이 띄워진 것이고, "close"인 경우 닫힌것을 의미합니다.

remarks

없음.
