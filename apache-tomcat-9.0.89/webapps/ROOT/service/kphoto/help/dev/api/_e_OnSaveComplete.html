K Photo :: OnSaveComplete

void RAONKPHOTO_OnSaveComplete(photoID, paramObj)

이미지 저장이 완료된 경우 발생합니다.

return value

없음.

parameters

photoID  이미지 저장을 완료한 포토의 id 입니다.
paramObj  저장된 이미지의 object 입니다.
- paramObj.height  이미지의 높이값을 의미합니다.
- paramObj.object  이미지를 의미합니다.
- paramObj.src  이미지가 저장된 경로를 의미합니다.
- paramObj.text  이미지에 추가된 텍스트를 의미합니다.
- paramObj.width  이미지의 넓이값을 의미합니다.

remarks

없음.
