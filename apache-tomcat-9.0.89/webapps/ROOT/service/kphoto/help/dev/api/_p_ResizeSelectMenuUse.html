K Photo :: Config :: ResizeSelectMenuUse

ResizeSelectMenuUse

크기 조절의 선택 탭을 표시하거나 숨기는 기능을 설정합니다.

remarks

기본값은 "1" 이고, "0" 으로 설정 시 크기 조절의 선택 탭을 표시하지 않습니다.

ResizeSelectMenu   크기 조절의 선택 탭에 들어갈 값을 설정합니다.
입력할 값은 '사이즈|기준' 으로 사이즈는 설정 할 사이즈 값이며, 기준 값은 w=가로, h=세로, l=긴축

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토의 크기조절 선택 탭을 표시합니다.
            ResizeSelectMenuUse: '1',

            // 포토의 크기조절 선택 탭에 들어갈 값을 넓이 300, 400, 높이 400, 500, 긴축 1000으로 설정합니다.
            ResizeSelectMenu: '300|w,400|w,400|h,500|h,1000|l'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
