K Photo :: Config :: StatusBarUse

StatusBarUse

포토의 상태바를 표시하거나 숨깁니다.

remarks

기본값은 "1" 이고, "0"으로 설정 시 상태바를 표시하지 않습니다.

sample code

<script type="text/javascript" src="kphoto/js/raonkphoto.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        var paramObj = {
            Id: 'photo1',

            // 포토의 상태바를 표시하지 않습니다.
            StatusBarUse: '0'
        };

        new RAONKPhoto(paramObj);
    </script>
</div>
