DEXT5 Editor :: KeepImageOriginalSizeAutoCheck

XML : <keep_image_original_size_auto_check>.....</keep_image_original_size_auto_check>
Config : config.KeepImageOriginalSizeAutoCheck

Config 설정이 우선 적용됩니다.
이미지 삽입 다이얼로그에서 파일 추가 시 원본크기 유지 항목의 자동체크 여부를 설정합니다.

remarks

기본값은 "0" 이며 사용 안함,
"1" 로 설정 시 이미지 삽입 다이얼로그에서 파일 추가 시 원본크기 유지 항목을 자동으로 체크 합니다.

sample code

이미지 삽입 다이얼로그에서 파일 추가 시 원본크기 유지 항목을 자동으로 체크 합니다.

XML

.....
<setting>
    <keep_image_original_size_auto_check>1</keep_image_original_size_auto_check>
    .....
</setting>
.....
