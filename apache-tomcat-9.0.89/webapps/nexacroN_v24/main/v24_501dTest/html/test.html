샘플 HTML 페이지

이 페이지는 HTML 문서의 기본 구조를 소개하는 샘플 페이지입니다.

1. 텍스트 추가하기

HTML에서는 다양한 방식으로 텍스트를 추가할 수 있습니다.

예시로, <h1> 태그는 가장 큰 제목을 나타내며, <h2> 태그는 두 번째 크기의 제목을 나타냅니다.

2. 목록 만들기

HTML에서는 순서가 있는 목록(<ol>)과 순서가 없는 목록(<ul>)을 만들 수 있습니다.

순서가 없는 목록:

  • HTML 기초 배우기
  • 텍스트 태그 사용하기
  • 이미지 삽입하기
  • 링크 추가하기

순서가 있는 목록:

  1. HTML 파일 만들기
  2. HTML 요소 추가하기
  3. 스타일링 적용하기

3. 테이블 만들기

HTML에서는 <table> 태그를 사용하여 표를 만들 수 있습니다. 표는 <tr>로 행을, <td>로 셀을 정의합니다.

이름 나이 직업

김철수 25 디자이너

4. 텍스트 강조하기

HTML에서는 <b>나 <strong> 태그를 사용하여 텍스트를 강조할 수 있습니다.

결론

이 페이지는 HTML을 사용하는 기본적인 방법들을 설명한 샘플 페이지입니다.
