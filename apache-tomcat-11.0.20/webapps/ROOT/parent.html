C 페이지 - iframe 연결

안내 사항

이 페이지는 iframe 스크롤 전파 테스트를 위한 C 페이지입니다.

iframe 내부 스크롤이 끝에 도달하면 이 페이지의 스크롤이 활성화됩니다.

하단 영역까지 스크롤하여 동작을 확인하세요.

※ iframe 안의 콘텐츠를 끝까지 스크롤하면 부모 페이지(C.html)로 스크롤이 전파됩니다.
반대로 최상단에서 위로 스크롤하면 부모 페이지가 위로 스크롤됩니다.
